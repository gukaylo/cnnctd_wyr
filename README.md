# 18+ Would You Rather Telegram Bot

Telegram-бот на `python-telegram-bot`, который запускает раунды "Would you rather..." для групповых чатов и собирает голоса через inline-кнопки.

## Быстрый старт

1. Склонируй репозиторий или скопируй файлы на сервер.
2. Создай окружение Python 3.10+ и установи зависимости:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Создай файл `.env` на основе `config.example.env` и задай `BOT_TOKEN=...` от BotFather.
4. Запусти бота:
   ```bash
   python main.py
   ```

## Использование

- `/start` или `/wyr` — запускает новый раунд голосования.
- Бот отправляет вопрос с кнопками выбора (A или B).
- Как только кто-то проголосовал, его имя появляется в том же сообщении с вопросом.
- После двух голосов в том же сообщении появляется предупреждение "⚠️ Осталось 20 секунд!".
- Через 20 секунд голосование завершается, и результаты (проценты и список участников по каждому варианту) выводятся в том же сообщении.

## Устройство

- `main.py` — логика бота, хранение активных раундов по `message_id` и таймеров.
- `questions.18` — 300 заранее заданных пар вопросов 18+.
- `config.example.env` — образец конфигурации.

Все данные хранятся в оперативной памяти, что позволяет поддерживать несколько чатов параллельно. После завершения раунда память очищается.
