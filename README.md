# 18+ Would You Rather Telegram Bot

Асинхронный Telegram-бот на `aiogram 3`, который запускает раунды "Would you rather..." для групповых чатов и собирает голоса через inline-кнопки.

## Быстрый старт

1. Склонируй репозиторий или скопируй файлы на сервер.
2. Создай окружение Python 3.10+ и установи зависимости:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Создай файл `.env` на основе `config.example.env` и задай `BOT_TOKEN=...` от BotFather.
4. Запусти бота:
   ```bash
   python main.py
   ```

## Использование

- `/start` в личке — краткая инструкция о добавлении бота в группу.
- В группе доступны команды `/would_you_rather`, `/wyr`, `/18` и каждый запуск создаёт независимый раунд.
- Бот показывает под вопросом список ников уже проголосовавших (без раскрытия выбранного варианта); при новых голосах сообщение обновляется.
- После двух уникальных голосов в том же сообщении появляется предупреждение "⚠️ Осталось 20 секунд!".
- Через 20 секунд голосование завершается, и результаты (проценты и список участников по каждому варианту) выводятся в том же первом сообщении.

## Устройство

- `main.py` — логика бота, хранение активных раундов по `message_id` и таймеров.
- `questions.18` — 300 заранее заданных пар вопросов 18+.
- `config.example.env` — образец конфигурации.

Все данные хранятся в оперативной памяти, что позволяет поддерживать несколько чатов параллельно. После завершения раунда память очищается.
